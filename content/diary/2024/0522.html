Ubuntu Composerのメモ

前提

  • ubuntuバージョン：22.04.4

バージョン確認

            lsb_release -a

実行結果

        
            Distributor ID: Ubuntu
            Description:    Ubuntu 22.04.4 LTS
            Release:        22.04

出てきた問題

phpをインストールし、2024/5/22最新のComposer(v 2.7.6)をインストールし、使おうとしたときphpバージョンが古くエラーが出てきた。

解決方法

ubuntu22.04.04は標準でphp8.1がインストールされてしまうようです。

なので、バージョンを指定してComposerに対応したバージョンをインストールする必要がある。

レポジトリの追加

            sudo add-apt-repository ppa:ondrej/php

パッケージリストを最新の状態に

        sudo apt update

phpのバージョンを指定してインストール(*は任意に変更)

        sudo apt install php8.*

私の場合はこれで実行できるようになったのでメモです。
