Reactとは

Remixや前回インストールしたものについて軽く説明しておこうと思います。


ReactとはUI構築のためのJavaScriptライブラリです。

Node.jsとはサーバー上でのJavaScript実行環境です。

タイトルにあるRemixはReactのスタックフレームワークです。

そのため、本題に入る前にフレームワーク元を触って勉強していこうと思います。

Reactのチュートリアルはこちらからご覧いただけます。




プロジェクトの作成

チュートリアルではCodeSandboxというエディタで行っていますがせっかくなので自分でプロジェクトを作るところからやっていきます。


プロジェクトの作成

            npx create-react-app プロジェクト名

私の場合、npmのバージョンが古いと警告が出たので、最新バージョンに更新してから行いました。

        npm install -g npm@latest

ローカルで実行するので脆弱性等は置いておきます。


プロジェクトのディレクトリに移動

        cd プロジェクト名

ローカルで実行

        npm start

http://localhost:3000/にブラウザでアクセスしてReactのロゴのページが表示されたら成功です。



Reactプロジェクトの作成から実行までを行いました。
